# Roadmap

Future ideas for the PCB portfolio:

- Component search from BOM CSV.
- Net search from exported netlist.
- Schematic-to-layout cross-links.
- Private project access.
- Revision comparison.
- STEP/3D viewer.
- Automatic hotspot generation from exported schematic metadata if available.
- ODB++ parsing exploration.
- Test log ingestion.

These are future features. The MVP intentionally focuses on static Altium exports, manually defined hotspots and callouts, and wiki-style engineering case studies.
