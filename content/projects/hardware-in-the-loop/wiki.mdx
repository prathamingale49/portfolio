# Hardware In The Loop

## Purpose

This project is a hardware-in-the-loop avionics board built around exported Altium schematic artifacts. The available PDF contains 30 schematic pages covering the top-level architecture, input power, load switching, boost conversion, CM5 control, ADC/DAC paths, sensor emulation, valve outputs, detonator interfaces, and output power.

## System Context

The board sits in a test and validation context for avionics hardware. It appears to combine power rails, compute/control, communications, analog conversion, and emulated loads or sensor interfaces so flight electronics can be exercised before full system integration.

## Architecture

The schematic PDF includes sheets for input power, input load switching, 48 V boost conversion, 5 V and 3.3 V regulation, CM5 main control, temperature sensing, communication, GPIO, ADC, DAC, differential sensors, thermocouple, current loop, RTD, valve, detonator, RBF tag, signal connectors, and output load switching.
