# Darcy Umbilical

## Purpose

The Darcy Umbilical project documents the connector boundary between ground-side support hardware and the vehicle-side interface. The available Altium export contains two schematic pages: `Ground_Side.SchDoc` and `Vehicle_Side.SchDoc`.

## Requirements

- Preserve the exported PDF as the schematic source for portfolio browsing.

## System Context

Umbilical hardware is a high-visibility interface in test and launch operations. The useful portfolio story here is pinout clarity, signal ownership, connector strategy, and how the board supports safe integration between ground equipment and vehicle electronics.

## Architecture

The schematic package is split into ground-side and vehicle-side sheets. This structure is useful for checking signal direction, connector pin assignments, and how test/support signals are handed across the umbilical boundary.
